Sobre o MyManga

MyManga é um projeto acadêmico de E-commerce Full Stack desenvolvido para demonstrar conceitos modernos de desenvolvimento web, incluindo autenticação, autorização, gerenciamento de estado e integração entre frontend e backend.

Backend

  • Java 17+ - Linguagem de programação
  • Spring Boot - Framework principal
  • Spring Security - Autenticação e autorização
  • JPA/Hibernate - Persistência de dados
  • PostgreSQL - Banco de dados relacional
  • JWT - Tokens de autenticação
  • Swagger/OpenAPI - Documentação da API

Frontend

  • Angular 18+ - Framework SPA
  • TypeScript - Linguagem tipada
  • Bootstrap 5 - Framework CSS
  • Bootstrap Icons - Biblioteca de ícones
  • RxJS - Programação reativa
  • Standalone Components - Arquitetura moderna
  • Guards & Interceptors - Segurança e HTTP

Funcionalidades

  • Sistema completo de autenticação e autorização
  • Cadastro de usuários com ativação por e-mail
  • Recuperação de senha
  • Catálogo de mangás com busca e paginação
  • Carrinho de compras com persistência
  • Sistema de pedidos
  • Painel administrativo para gerenciar mangás
  • Gerenciamento de volumes e estoque
  • Visualização de todos os pedidos (admin)
  • Sistema de avaliação (rating)
  • Design responsivo para mobile
  • Documentação completa da API com Swagger
